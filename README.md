# # vue-knober

VueKnober - a knob ui component

![vue-knobe](https://raw.githubusercontent.com/shiasn/vue-knober/master/docs/images/vue-knober.gif)

## Usage

| prop | describe | default |
| --- | --- | --- |
| size | define knob size | 250(px) |
| value | v-model | 0 |
| min | minimum value | 0 |
| max | maximum value | 255  |
